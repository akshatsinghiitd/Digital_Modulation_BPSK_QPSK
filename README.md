# Digital Modulation Analysis using BPSK and QPSK over AWGN Channel

##  Project Overview
This project presents a MATLAB-based simulation and performance analysis of **BPSK** and **QPSK** digital modulation schemes over an Additive White Gaussian Noise (AWGN) channel.

The performance of the system is evaluated in terms of **Bit Error Rate (BER)** for different Signal-to-Noise Ratio (SNR) values.

---

##  Objectives
- Implement BPSK and QPSK modulation techniques in MATLAB
- Model the effect of AWGN channel
- Perform demodulation using hard decision detection
- Analyze BER performance versus SNR
- Visualize noise impact using constellation diagrams



##  System Model
The communication system consists of:
1. Random Bit Generator  
2. BPSK / QPSK Modulator  
3. AWGN Channel  
4. Demodulator  
5. BER Analyzer
